# ph-perfume-price-finder
Got tired of opening five different Google Sheets every time I wanted to check who had the best price on something, so I built a single page that pulls all of them together and lets you search, filter, and sort across everything at once.

What it pulls from right now:


ODYSSEY MNL
Perfumejunkieph
Alloriginal100%
Perfumania by Pao
Que Bros


That's 2,600+ listings across 248 brands in one searchable view, updated live from each seller's sheet.

What it actually does:


🔍 Instant search — type a brand, name, or even a nickname (e.g. "CDN," "Khamrah," "Hawas") and it matches across all 5 sheets at once
💰 Budget slider (₱500–₱30,000) to cap your search to what you're actually willing to spend
↕️ Sort by best match, lowest price, highest price, or best ₱-per-mL (so you can actually compare value, not just sticker price)
🏷️ Filter by condition — BNIB/New, Tester, Partial, or Other
🏪 Filter by store — isolate one seller or compare across all of them
❤️ Wishlist — heart anything interesting to shortlist it for that session
🌗 Light/dark mode
🔄 One-click sync — refresh all 5 sheets live to get current prices instead of a stale snapshot
💾 Save synced copy — download a fresh offline copy with that day's prices baked in, so you're not re-syncing every time you open it
Every result links straight to the seller's original sheet so you can message them and order — this is just a search/comparison layer on top of their public price lists, not a storefront itself


How to use it:


Open the tool: [LINK HERE]
Search for what you're hunting, or just browse using the filters
Drag the budget slider and tap condition chips to narrow things down
Hit ♥ on anything you want to track, check your wishlist tab later
Click "Open sheet" on a result to go message that seller directly
Hit Sync prices anytime you want the latest numbers across all 5 sheets, and Save synced copy if you want an offline copy with today's prices locked in


Some honest caveats:


This is a comparison tool over publicly shared seller price sheets — I don't run any of these stores, and all the actual buying/selling happens with the original seller
Prices shift often, so hit Sync before making a decision if it's been a while
Wishlist is session-only right now — it clears if you close the tab (might add saving later if there's interest)
Only 5 sellers for now — if you know other PH resellers who keep an updated public price sheet, drop a link and I'll look at adding them


Open to feedback, bug reports, and "hey can you add X seller" requests. Built this for myself first, figured it might save other people some tab-switching too.
